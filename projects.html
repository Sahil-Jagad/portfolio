Should I Go

Should I Go is a health and saftey app designed and developed at UCI's premier hackathon Hack @ UCI. My team and I wanted to develop an application that tells users wether the symptoms they have are in line with the CDC's definition of COVID-19 as well as display local COVID-19 statistics to help the user make an informed decision wether to go outside. I developed the front end of the application using Figma and React.js. On the backend I wrote a NLP script using NLTK Library that tokenized user imputed symptoms and utilized a cosine similarity algorithm to predict if user has COVID-19. This project won Best Health App at Hack @ UCI.

Zot N' Seek

Zot N Seek was designed and implemeted at UCI ICS Student Council's Web Jam 2020. The idea behind the project was to create a simple game that allows UCI students and professors who were unable to expirience UCI thier first year due to Covid-19. I helped in the backend development of this project primarily by building a simple API with SQL and Heroku linking locations with Google Maps street views. I also helped design the game logic and the home page front-end. The project won first place at the WebJam.
